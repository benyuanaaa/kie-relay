如何调用？

Python:

import openai client = openai.OpenAI( base_url="https://qidianai.indevs.in/v1", api_key="你的API Key" ) resp = client.images.generate( model="z-image", prompt="一只白色卡通熊", n=1, size="1024x1024" ) print(resp.data[0].url)

curl:

curl https://qidianai.indevs.in/v1/images/generations \ -H "Authorization: Bearer 你的API Key" \ -d '{"model":"z-image","prompt":"白色卡通熊","size":"1024x1024"}'
